# Лабораторная работа №1 (вариант 19)

Решение СЛАУ методом **Гаусса с выбором главного элемента по столбцу** на JavaScript (Node.js).
Дополнительно выполняется расчёт через библиотеку **mathjs** для сравнения решения и определителя.

```bash
npm start
```

## Формат входного файла

Поддерживаются два формата.

### Формат 1 (расширенная матрица `A|B`)

1. Первая строка: `n`
2. Далее `n` строк, в каждой `n + 1` чисел:

```text
3
2 1 -1 8
-3 -1 2 -11
-2 1 2 -3
```

### Формат 2 (матрица `A` и отдельный вектор `B`)

1. Первая строка: `n`
2. Далее `n` строк по `n` чисел (матрица `A`)
3. Последняя строка: `n` чисел (вектор `B`)

```text
3
2 1 -1
-3 -1 2
-2 1 2
8 -11 -3
```

Пустые строки и строки, начинающиеся с `#`, игнорируются
